## 🌈 주요기능
- 홈페이지 : 로그인/ 회원가입 페이지로 이동함.
- 로그인/회원가입 : 정규식을 이용하여 제한을 둠. 아이디/비번/닉네임 기입
- 메인페이지 : 지도, 업체 목록, 북마크
- 상세페이지 : 업체 정보, 리뷰
- 마이페이지 : 북마크, 리뷰, 프로필
